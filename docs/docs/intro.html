Getting Started

Once PYroMat is installed, the package should be available at the command line as pyromat. Once it is imported, the get function will retrieve objects that represent individual species. For example, let's look at diatomic nitrogen.

>>> import pyromat as pyro >>> N2 = pyro.get('ig.N2')

The argument to the get function is the species ID. It is comprised of two parts: the chemical compound, "N2," and the data collection to which it belongs, "ig," for "ideal gas."
