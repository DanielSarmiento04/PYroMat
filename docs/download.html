PYroMat

Downloads

PYroMat is available for download from Python's Package Index, Github and Sourceforge. PYroMat was developed to support both Python2 and Python3.

If you download a version later than 1.3 from Github or Sourceforge, the root directory will also contain test.py, which generates a .log file showing the results of tests of each of PYroMat's classes.

The python package index

The PYroMat package index can be found here. To install from a command line, run the following command as an administrator:

pip install pyromat
To upgrade from an older version
pip install pyromat --upgrade

Github

The PYroMat package index can be found at https://github.com/chmarti1/PYroMat. To get your own working copy, fire up your command line:
git clone https://github.com/chmarti1/PYroMat.git
cd PYroMat
python setup.py install
The last command may need to be executed as an administrator on your system.

Sourceforge

The PYroMat sourceforge page can be found at https://sourceforge.net/projects/pyro/. The sourceforge distributions of PYroMat are .zip, .bz2, and .gzip files. Choose one, download it, and extract it. In the root directory, fire up your command line:
python setup.py install
