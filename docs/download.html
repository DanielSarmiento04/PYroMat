PYroMat

Downloads

PYroMat is available for download from Python's Package Index, Github and Sourceforge.

The Python Package Index

The PYroMat package index can be found here. To install from a command line, run the following command as an administrator:
pip install pyromat
To upgrade from an older version
pip install pyromat --upgrade

Github

The PYroMat package index can be found here. To get your own working copy:
git clone https://github.com/chmarti1/PYroMat
