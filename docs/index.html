PYroMat

Thermodynamic Computational Tools
for Python

PYroMat is a flexible platform for calculating thermodynamic properties in Python.

import pyromat as pyro
S = pyro.get('steam')
# Calculate enthalpy as 450K and 3bar
h = S.h(T=450., p=3.)
# Go backwards
T = S.T_h(h=h, p=3.)

Features

Version 1.4.1 of PYroMat contains 79 substances including imperfect gas mixtures and steam. The data come from different sources and they are in different formats, but it all looks the same to the user!

import pyromat as pyro
# Ideal gases
N2 = pyro.get('N2')
nitrogen_enthalpy = N2.h(T=792.)
# 528.18344232751906 kJ/kg

# Perfect gases
Ar = pyro.get('Ar')
argon_enthalpy = Ar.h(T=792.)
# 256.96512391640385 kJ/kg

# Mixtures
air = pyro.get('air')
air_enthalpy = air.h(T=792.)
# 510.77100446306434 kJ/kg

# multi-phase
S = pyro.get('steam')
steam_enthalpy = S.h(T=792.)
# 3529.0511381775596 kJ/kg

As of version 1.4.1, PYroMat includes multi-phase data for steam with refrigerants on the way. New inverse properties like T_s() and T_h() let users invert properties for cycle analysis.

Interface

When a user wants to work with a substance, the user requests an object that represents that substance (like nitrogen, argon, air, and steam above). These objects have their own methods for calculating enthalpy, entropy, density, specific volume, molecular weight, internal energy, and others from temperature and pressure.

Numpy compatibility means that PYroMat interfaces can handle arrays and iterables seamlessly.

import numpy as np
import matplotlib.pyplot as plt
T = np.arange(300.,5000.,100.)
plt.plot(T, air.cp(T))

For a more extensive idea of what PYro can do, check out the examples.
